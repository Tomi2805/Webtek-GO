# Webtek-GO
Repository for prosjekt i IT2805 - nettside for Trondheim Go-klubb.

Alt under folderen <i>public_html</i> skal kunne kopieres rett til folderen med samme navn på folk.ntnu.no-kontoen vi bruker når vi legger ut.

Jeg la til noen filer og subfoldere iht. beskrivelsene i ProjectRoadMap-dokumentet.

Eventuelle filer som ikke er del av nettsiden vi lager, eller er dokumentasjon for faglærer (dvs. de som er under <i>public_html/Documents</i>), kan vi legge på samme nivå som denne tekstfilen du leser nå eller i en annen subfolder enn <i>public_html</i>.
